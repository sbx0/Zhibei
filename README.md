### [智贝](http://zb.sbx0.cn "智贝")  
  
###### 立足于技术商业化  
###### 基于Spring Boot和Vue.js的JavaWeb项目  
  
------------  
  

#### 已完成特性  
- 通用化后台  
只需简单配置，即可对新增表进行增删改操作。  
- 多语言支持  
支持简繁英自由切换。  
- 大文件上传  
支持大文件上传，秒传。  
- 模块化界面  
减少开发负担。  
- 权限全控制  
几乎所有操作都可以自定义控制授权。  
- 日志全记录  
几乎所有操作记录在案，可查可追溯。
